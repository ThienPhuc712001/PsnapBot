# 🚀 PSnapBOT - Optimized Commands

## 🎯 Quick Start (Chỉ cần 1 lệnh)

```cmd
cd dev_agent
WORK
```

## 📋 Danh sách lệnh tối ưu

### 🔥 Phổ biến nhất
```cmd
WORK        # Menu làm việc đầy đủ
CHAT        # Bắt đầu chat với PSnapBOT
FIX         # Sửa lỗi tự động
ADD         # Thêm tính năng mới
```

### 🎯 Lệnh theo chức năng

#### **1. Giao tiếp với PSnapBOT**
```cmd
CHAT                    # Chat mode tương tác
WORK                    # Menu làm việc đầy đủ
PSnapBOT                # Menu chuyên sâu
```

#### **2. Phát triển & Sửa lỗi**
```cmd
FIX "build error"        # Sửa lỗi build
FIX "test failure"       # Sửa lỗi test
ADD "user login"         # Thêm feature đăng nhập
ADD "API endpoint"       # Thêm API endpoint
```

#### **3. Phân tích & Tối ưu**
```cmd
WORK → chọn 4           # Phân tích code
WORK → chọn 5           # Chạy tests
WORK → chọn 6           # Tạo docs
```

#### **4. Hệ thống**
```cmd
WORK → chọn 7           # Custom command
WORK → chọn 8           # Settings
PSnapBOT → chọn 7       # System status
```

## 🎪 Cách sử dụng thực tế

## 🆘 Nếu có lỗi

```cmd

```

### **Cần reset:**
```cmd

```

### **Cập nhật API key:**
```cmd

```

## 📝 Tips

1. **Luôn bắt đầu với `WORK`** - Menu dễ dùng nhất
2. **Dùng `CHAT`** cho các tác vụ phức tạp cần trao đổi
3. **Dùng `FIX`** cho các lỗi phổ biến
4. **Dùng `ADD`** cho phát triển feature mới
5. **Mọi lệnh đều tự động lưu progress**
